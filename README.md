年齢制限：１１歳以上

# ストーリー

天下布武を成し遂げた戦国大名、織田信長。
あなたは彼が「うつけ」と呼ばれるころから付き従う忠実な家臣だ。
主君の野望を助けるため、軍団を率い、城を落とし、最も手柄を立てよう。

# 勝利条件

ゲーム終了時に、自分が所有するカードの勝利点の合計値が最も高いプレイヤーが勝者となります。点数が同じであれば、官位「第六天魔王」を持つプレイヤーが勝利します。

# ゲームの準備

![](https://i.imgur.com/5YlPC5X.jpg)

## 武将カード

- 武将カードから**影武者**を表向きに取り除き、裏向きにして良く切ります。
- 各プレイヤーに武将カードを表向きに1枚づつ配ります。

## 城カード

- 城カードから**本能寺**を取り除き、裏向きにして良く切ります。

## 地図、作戦、官位カード

- 地図カードを番号順に一列に並べます。地図カードと同じエリア名（東１、西２など）を持つ官位カードを、地図カードの上に並べます。「源氏の長者」を「北米奉行」の右に並べます。
- 地図カードと同じダイス目を持つ作戦カードを地図カードの下に並べます。合戦（６）の右隣に、籠城戦と記載された作戦カードを並べます。

## 開始プレイヤー

- 適当な方法で開始プレイヤーを決定します。これが2回目のゲームであれば、前回の勝利者が開始プレイヤーになります。最初の手番では、開始プレイヤーは「開始プレイヤーカード」によるボーナスダイスを取得しません。

# 手番の進行

手番は以下の順序で進行します。

1. 準備フェイズ
2. 謀略フェイズ
3. 行軍フェイズ
4. 調略フェイズ
5. 合戦フェイズ
6. 内政フェイズ

1～6の手番によりで1ターンが終了します。ゲーム終了までターンを繰り返します。

# ゲームの終了

プレイヤー人数に応じたターン数実施後ゲームは終了します。
ターンマーカーを作戦カードの上に置き、現在のターンを表示します。

|プレイヤー人数|ターン数|
|:--|:--|
|2名～3名|5ターン|
|4名～6名|4ターン|

# １．準備フェイズ

## ダイス（兵力）の配布

各プレイヤーは、ターン数＋１個のダイスを受け取ります。前のターンに「開始プレイヤー」カードを受け取っていれば更に＋１個、「武田信玄」カードを持っていれば更に＋１個のダイスを受け取ります。

## カードの配置

### 城

城カードの山札から、4枚のカードを引いて、「合戦」と書かれた３～６番の作戦カードの下に表向きに配置します。終了ボーナスカードの下に、指定された枚数の城カードを表向きに配置します。

### 武将

武将カードの山札から、2枚のカードを引いて、「調略」と書かれた１番～２番の作戦カードの下に表向きに配置します。終了ボーナスカードの下に、指定された枚数の武将カードを表向きに配置します。

### 開始プレイヤー

開始プレイヤーカードを終了ボーナスカードの下に戻します

## 最終ターンの場合

本能寺カードを、アクションカード「包囲戦」の山札に置きます。

# カード入手時の処理

武将カード、城カードを入手した場合、以下の処理が発生します。

## 武将カード

### 源氏の長者

武将カードを入手後、武将カードの枚数が全プレイヤーの中で最大である場合、官位カード「源氏の棟梁」を得ます。単独で最大である必要はなく、他に同じ枚数の武将カードを持っているプレイヤーがいたとしても、後手番が優先されます。

ただし、ゲームの準備時に配られた武将カードの入手では発生しません。

### 領地能力

「領地」能力を持つ武将（伊藤マンショ、九鬼義隆）を入手した場合、マップから一つを選び、領地マーカーを一つ置きます。領地：東とあれば、東１、東２、東３に、領地：西とあれば、西１、西２、西３のマップにマーカーを置くことができます。これらの武将は、入手時以外に内政フェイズにも領地マーカーを配置することができます。

ゲームの準備時に配られた武将カードが領地能力を持つ場合、領地マーカーの配置は行います。


## 城カード

城カードを入手した場合、記載されたマップ名と同じマップカード上に、城カードのダイス数と同じ個数の領地マーカーを置きます。ダイス目は関係なく、ダイスの個数を見ます（例：姫路城[2][3]を入手した場合、西１マップ上に領地マーカーを2個置きます）

城カードが「本能寺」であった場合、領地マーカー３つを、好きなマップ上に配置できます。これは同じマップであっても、異なる３つのマップであってもかまいません。

## 領地マーカー

領地マーカーを配置後、同じマップ内の領地マーカーの個数が全プレイヤーの中で最大である場合、マップ名に応じた官位カードを受け取ります。単独で最大である必要はなく、他に同じ個数の領地マーカーを配置しているプレイヤーがいたとしても、後手番が優先されます。

## カードを失った場合の処理

武将カードを失ったことによる、官位カードの移動や、領地マーカーの除去は発生しません。
城カードを失ったとしても、配置済の領地マーカーの除去は発生しません。

# ２．謀略フェイズ

一部の武将カードの特殊能力を使用します。

## 2.1 お市

ゲーム内で表向きに置かれた「本能寺」以外の城カードから１枚選んでお市のカードと交換することができます。城カードを入手した場合の処理を行います。

籠城戦アクションカードに積み重ねて置かれた城カードも全て対象にすることができます。

## 2.2 松永久秀

、松永久秀のカードと交換します。武将カードを入手した場合の処理を行います。

# ３．行軍フェイズ

開始プレイヤーから時計回りに、すべてのプレイヤーが手元のダイスを配置し終わるまで手順を繰り返します。手順を行っているプレイヤーを手番プレイヤーと呼びます。

## 3.1 ダイスロール

手番プレイヤーは手元のダイスを全てロールします。

## 3.2 武将能力の使用

武将カードの下に、3/4->2や1/2->5のように記載されている場合、その武将はダイス変換能力を持ち、ロールしたダイスの目を変更できます。

・同じ出目は全て変更されます。一部だけ変更することはできません（例：3,3,3,4の出目に対し、3->1の能力を使用した場合、必ず1,1,1,4になります）

・複数の武将のダイス変換能力を同一のロールに使用することはできません。

・1/2->6とある場合、1の出目を6にするか、2の出目を6にするか、どちらか一方のみ行えます。1と2の目をどちらも6にすることはできません。

・1->5/6とある場合、1の出目を5にするか、1の出目を6にするか、どちらか一方を行えます。


## 3.3 作戦カードへのダイスの配置

プレイヤーは1種類の出目を選んで、その出目のすべてのダイスを、出目に対応した作戦カードの下に配置します。必ず一つ以上のダイスを配置しなくてはなりません。

作戦カードのうち、籠城戦カードだけは、1の出目、6の出目のどちらでも配置できます。ただし、同時に配置できるのは1種類の出目に限られます（例：1,1,6,6の出目を全て籠城戦に配置することはできません）

# ４．調略フェイズ

調略のアクションカード、１→２の順に解決を行います。配置したダイスの数を比べ、最も多いダイスを置いたプレイヤーが勝者となり武将カードを得ます。ダイスの数が同じである場合、最も最後にダイスを置いたプレイヤーが勝者になります。

# ５．合戦フェイズ

合戦のアクションカード３から６の順に解決を行います。その後、籠城戦のアクションカードの解決を行います。

配置したダイスの数を比べ、最も多いダイスを置いたプレイヤーがそのアクションカードの総大将になります。ダイスの数が同じである場合、最も最後にダイスを置いたプレイヤーが総大将になります。

総大将になったプレイヤーは、他のプレイヤーが置いたダイスを含め全てロールします。この際、武将のダイス変換能力を使用することができます。

出目を組み合わせて、城カードに記載されたダイスと同じ並びにすることができれば、城カードを入手することができます。（例：小田原城 4/5/6に対して、ダイスを6個振った、1,1,1,1,4,5であったが、弥助の能力で4,5,6,6,6,6に変換した。城と同じ4/5/6の組を作ることができたので、城カードを入手する）

## 籠城戦

籠城戦のアクションカードには、複数の城カードが置かれている場合があります。ここに積み重ねて置かれた城カードは公開情報であり、どのプレイヤーもいつでも参照できます。総大将のプレイヤーは、その中から１枚だけ選んで、合戦フェイズと同じ処理を行います。

## 合戦フェイズに影響する武将の特殊能力

### 上杉謙信

総大将になるたびに、追加ダイスを２個得ます。

### 足利義昭、本願寺顕如、竹中半兵衛

矢印の右が×になっている能力を持ちます。対象の城が持つダイス目について、一致するものをどちらか一つ、無いものとして判定できます。その結果、ダイス目がなくなった場合、判定なしで城カードを入手できます。（例：法王庁は1/1/1のダイス目を持ちます。足利義明の能力1->Xを使用した場合、法王庁はダイスなしになり、判定なしで入手できます）

# ６．内政フェイズ

## 未取得の城カード、武将カードの整理

調略、合戦のアクションカード（番号１～６）に置かれたままで、このターンプレイヤーに入手されていない城カード、武将カードがあれば、籠城戦アクションカードの山札に表向きに積み重ねて加えます。

## 領地能力

領地能力を持つ武将がいれば、マップを一つ選んで領地マーカーを一つ置きます。
